## 1. Estructura de carpetas

- **src**: Carpeta destinada a los archivos de código fuente (.py).
- **doc**: Espacio para la documentación del sistema.
- **bin**: Directorio para ejecutables del proyecto.
- **script**: Directorio para scripts del proyecto.

## 2. Configuración del Entorno

Para ejecutar los archivos de este repositorio, siga estos pasos:

1. Clonar el repositorio en su máquina con:
   ```bash
       git clone https://github.com/MaitenBlanc/UADER_IS2_BLANC.git
   ```
2. Asegurarse de tener instalado Python 3 o superior.
3. Instalar las dependencias necesarias mediante pip:
   - Matplotlib (para los gráficos de Collatz).

## 3. Listado de tecnologías principales
1. **Git**: Para el seguimiento de cambios.
2. **Python**: Lenguaje principal de desarrollo.
3. **Markdown**: Para la documentación técnica.
4. **Matplotlib**: Para la generación de visualizaciones.
